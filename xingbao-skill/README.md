# README

![星宝 Banner](https://cdn.gooo.ai/assets/banner-xingbao.png)

一个结合命理（八字）、星座（占星）、性格分析的成长陪伴 AI Skill，帮助用户在职场转型、人际冲突、情感困惑时获得深度自我认知，并以温暖朋友的方式陪伴行动落地。

### 🔮 命理分析

- ✅ 四柱八字排盘

- ✅ 五行分布统计

- ✅ 十神配置分析

- ✅ 日主强弱判断

- ✅ 用神喜忌推断

### ⭐ 星座占星

- ✅ 太阳/月亮/上升星座计算

- ✅ 行星宫位分析

- ✅ 相位关系解读

- ✅ 事业/感情宫位解析

### 🧠 性格整合

- ✅ 命理+星座→MBTI 映射

- ✅ 大五人格维度分析

- ✅ 个性化沟通风格适配

### 💬 成长陪伴

- ✅ 多轮目标澄清对话

- ✅ 3 个月行动计划制定

- ✅ 周度行动清单拆解

- ✅ 长期陪伴追踪机制
