# 八字命理基础指南

## 什么是八字命理？

八字命理，又称「四柱推命」，是根据一个人出生的年、月、日、时，推算出四组天干地支（共八個字），据此分析人的命运、运势与性格的传统学问。

---

## 核心概念

### 四柱

| 四柱 | 组成 | 代表含义 |
| --- | --- | --- |
| **年柱** | 年干 + 年支 | 祖上根基、童年环境 |
| **月柱** | 月干 + 月支 | 父母宫位、青年运势 |
| **日柱** | 日干 + 日支 | **核心（日主）**、中年运势 |
| **时柱** | 时干 + 时支 | 子女宫位、晚年运势 |

## 五行生克

### 相生关系

```plaintext
木 → 火 → 土 → 金 → 水 → 木
```

- 木生火：木燃烧产生火

- 火生土：火焚烧后成灰为土

- 土生金：土中蕴含金属矿物

- 金生水：金属受热可化为液体

- 水生木：水滋润树木生长

### 相克关系

```plaintext
木 → 土 → 水 → 火 → 金 → 木
```

- 木克土：树木根系穿透土壤

- 土克水：土可以阻挡水流

- 水克火：水可以扑灭火

- 火克金：火可以熔化金属

- 金克木：金属工具可以砍伐树木

## 十神系统

十神是根据日主（日干）与其他干支的关系确定的，代表不同的人生面向和心理特质。

### 十神定义

| 十神 | 定义 | 代表含义 |
| --- | --- | --- |
| **比肩** | 同我且同阴阳 | 自我、独立、主见、竞争 |
| **劫财** | 同我且异阴阳 | 行动力、冒险、合作、破财 |
| **食神** | 我生且同阴阳 | 表达、享受、才华、温和 |
| **伤官** | 我生且异阴阳 | 创意、叛逆、口才、自由 |
| **偏财** | 我克且同阴阳 | 机遇、灵活、社交、投机 |
| **正财** | 我克且异阴阳 | 稳定、务实、责任、积累 |
| **七杀** | 克我且同阴阳 | 魄力、压力、挑战、权威 |
| **正官** | 克我且异阴阳 | 规则、名誉、责任、约束 |
| **偏印** | 生我且同阴阳 | 智慧、独特、思考、偏门 |
| **正印** | 生我且异阴阳 | 学识、贵人、保护、传统 |

## 日主强弱判断

### 得令

日主是否生于当令的季节（月支五行与日主相同或相生）

| 日主 | 当令月份 |
| --- | --- |
| 甲乙木 | 寅卯月（春季） |
| 丙丁火 | 巳午月（夏季） |
| 戊己土 | 辰戌丑未月（四季末） |
| 庚辛金 | 申酉月（秋季） |
| 壬癸水 | 亥子月（冬季） |

### 得地

日主在地支是否有根（地支中是否有同五行的藏干）

### 得势

四柱中是否有其他干支生扶日主

### 强弱分类

- **身强**：日主得令、得地、得势

- **身弱**：日主失令、失地、失势

- **中和**：各方面较为平衡

## 用神喜忌

### 用神

对命主最有利的五行，可以平衡八字、补救命局缺陷。

### 喜神

辅助用神的五行，增强用神的力量。

### 忌神

对命主不利的五行，会加剧命局的不平衡。

### 闲神

对命局影响不大的五行。

### 用神确定原则

- **身强**：宜泄（食伤）、宜克（官杀）、宜耗（财）

- **身弱**：宜生（印）、宜扶（比劫）
